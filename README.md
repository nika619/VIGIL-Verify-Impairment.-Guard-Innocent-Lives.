## How to Run Locally

1. **Clone & Install**
   ```bash
   git clone https://github.com/nika619/VIGIL.git
   npm install
   ```

2. **Configure API Key**
   Create a `.env` file in the root and add your Gemini API Key:
   ```env
   GEMINI_API_KEY=your_key_here
   ```

3. **Start the Engine**
   ```bash
   npm start
   ```
   Open `http://localhost:8080` to experience VIGIL.

## Design Philosophy

> *"Premium means frictionless. If the UI bothers the user, it has failed."*

VIGIL is designed to feel like a native instrument panel in a luxury electric vehicle. The two-phase flow (Calibration вҶ’ Assessment) ensures the operator is never overwhelmed, while the adaptive color theming and animated gauge provide immediate, intuitive feedback without requiring the user to read a single number.
